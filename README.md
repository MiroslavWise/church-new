Скачать проект

``` git clone https://gitlab.com/MiroslavWise/church-new.git```

Затем установить зависимости

``` npm i ``` || ``` npm install ```

Запустить проект

``` npm start ```


Добавить файлы в Git

``` git add . ```
``` git commit -m "комментарий" ```
``` git push ```

Стянуть

``` git pull ```
